# 谷歌浏览器调试JS

用控制台记录JS的结果, 什么类型都可以, 例如:

```
    var test = 'aaaa';
    console.log(test);
```

然后打开谷歌的调试平台, 选择 `Console`, 就会看到结果了. 
结果有颜色高亮. 可以展开和收起, 十分的方便. 

这个要比`alert()`方便多了. 
因为`alert()`会打断程序的执行. 而且结果也不能复制, 点确定后就没了.
